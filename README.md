This a application based on kivy run on android phone.

Before you use this code, make sure installed kivy.
# 安装python相关模块：
```
pip install kivy
pip install kivy_examples
pip install pymysql
pip install jieba
```
# 参考运行环境配置
**python3.6.9**
**kivy 1.11.1**
**ubuntu 18.04.1**
**mysql5.7.27**
# 运行注意事项
需要在mysql中创建一个用户名root，密码123456,允许所有权限;
